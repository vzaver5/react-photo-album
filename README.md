# React-photo-album

This project uses React/Vite to create a site that allows the user to upload photos and view the gallery using Supabase to hold the images.

Steps:
- Create .env.local
- VITE_SUPABASE_URL=[Add your URL here]
- VITE_SUPABASE_ANON_KEY=[Add your key here]

Only operational on localhost for now.
